Paramètres

Jours fériés

Dates marquées sur le calendrier par un trait pointillé rouge. Utile pour signaler que rien ne doit avancer ces jours-là.

    Plages temporelles à signaler — gel des changements, fermeture, vacances scolaires, vigilance accrue. Affichées comme bandes hachurées sur le Gantt.

      Dans la colonne Préc. du tableau, saisissez les ID des tâches prédécesseurs séparés par des virgules. Suffixez par un type pour autre chose que Finish-to-Start :

      • 3  → FS (Fin → Début, défaut)
      • 3SS  → Start-to-Start
      • 3FF  → Finish-to-Finish
      • 3SF  → Start-to-Finish
      • 3, 5SS, 7  → plusieurs prédécesseurs

      Les dépendances violées (prédécesseur incompatible) apparaissent en flèche rouge pointillée.
